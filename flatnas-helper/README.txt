================================================================
                FlatNas 专用助手 (FlatNas Helper)
                版本: v3.0
================================================================

【简介】
这是一个浏览器扩展插件，专配合 FlatNas 导航页使用。
它的作用是解除浏览器对 iframe 的安全限制，让你在 FlatNas 里能正常嵌入百度、淘宝、路由器后台等原本无法显示的网页。

为了安全起见，本插件仅在你设置的 NAS 地址下生效，不会影响其他网站的安全性。

----------------------------------------------------------------
【如何安装】
1. 解压你下载的这个压缩包（请勿直接删除解压后的文件夹）。
2. 打开 Chrome 或 Edge 浏览器。
3. 在地址栏输入: chrome://extensions (或者 edge://extensions) 并回车。
4. 打开页面右上角的【开发者模式】开关。
5. 点击左上角的【加载已解压的扩展程序】按钮。
6. 选择本插件所在的文件夹，安装完成！

----------------------------------------------------------------
【如何配置】(重要！)
安装完成后，你需要告诉插件你的 NAS 地址是多少：

1. 在浏览器右上角的插件栏（拼图图标）里，找到“FlatNas 专用助手”并点击。
2. 在弹出的窗口中，输入你的 NAS 访问地址。
   - 如果是内网，输入如: 192.168.1.100
   - 如果是外网域名，输入如: nas.myhome.com
   - 如果两者都要，用逗号隔开: 192.168.1.100, nas.myhome.com
3. 点击【保存并生效】。
4. 回到你的 FlatNas 页面，刷新网页，原来报错的 iframe 现在应该能正常显示了。
